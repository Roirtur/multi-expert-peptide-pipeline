\section{Conclusion}\label{sec:conclusion}
This report presented a multi-expert agentic pipeline for \textit{in silico} peptide design, where a generator, a chemistry constraints agent, and a biology scoring agent are coordinated by an iterative orchestrator. Across experiments, the system showed consistent optimization behavior: iterative rounds improved ranking quality, especially in harder regimes (notably short target lengths), while maintaining strong sequence-space exploration and substantially lower duplication than the non-agentic baseline.

Beyond final Top-$K$ outputs, the main contribution is methodological: explicit role decomposition, reproducible orchestration, and traceable decision flow at each round (generation, filtering, scoring, selection). This architecture makes trade-offs visible, supports controlled experimentation, and provides a practical framework for comparing optimization policies under shared proxy constraints.

At the same time, the biological signal used here remains a proxy and should not be interpreted as experimental evidence of efficacy. The current system is therefore positioned as a robust AI engineering proof of concept rather than a biological validation pipeline. Future work should prioritize peptide-specialized biological predictors, richer chemistry-aware descriptors, and broader ablations (e.g., removing one expert or iteration) to further quantify each component's contribution.

All source code, experiment scripts, configurations, and run logs are provided in the project repository \cite{multi_expert_repo} to support reproducibility and facilitate extension of this pipeline in subsequent work.
