# Multi-Expert Peptide Pipeline

A multi-expert agentic pipeline for in silico peptide generation. This PoC orchestrates AI, Chemistry, and Bio agents to design, filter, and score molecular candidates through iterative optimization loops.

### Planned Components

- **Pipeline Orchestrator** - Coordinates multi-agent workflow
- **AI Agent** - Generates peptide candidates
- **Chemistry Agent** - Filters and scores based on chemical properties
- **Bio Agent** - Scores based on biological activity predictions
- **Baseline Agents** - Random/heuristic baseline implementations

### Technology Stack (Planned)

- Python 3.9+
- Data processing: pandas, numpy
- Validation: pydantic, jsonschema
- Logging: Python logging with structured formats
